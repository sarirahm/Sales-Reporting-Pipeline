# Sales Reporting Pipeline

An end-to-end Data Engineering project that automates the extraction, transformation, and loading (ETL) of sales data from multiple sources into a centralized reporting dataset.

## Project Objectives

- Build automated ETL workflows
- Ensure data quality and consistency
- Generate analysis-ready datasets
- Support business reporting and decision-making
- Automate pipeline execution using Apache Airflow

## Tech Stack

- Python
- Pandas
- MySQL
- Apache Airflow
- Power BI
- Git & GitHub

## Architecture

CSV Files
    ↓
Python ETL
    ↓
MySQL Data Warehouse
    ↓
Apache Airflow
    ↓
Power BI Dashboard

## Key Features

- Automated ETL pipeline
- Data validation and cleansing
- Revenue calculation and aggregation
- Workflow scheduling with Airflow
- Reporting-ready data mart
- Interactive Power BI dashboard
